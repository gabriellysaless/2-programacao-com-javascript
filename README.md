# 💻 Programação com JavaScript

> **Repositório destinado aos estudos e exercícios desenvolvidos durante o módulo de Programação com JavaScript da EBAC.**

Este repositório reúne os projetos e exercícios realizados durante o módulo, abordando os principais conceitos da linguagem JavaScript, lógica de programação e manipulação do navegador.

O objetivo é documentar minha evolução no aprendizado, consolidar os fundamentos da linguagem e servir como material de consulta durante minha formação em Desenvolvimento Front-end.

## 🚀 Tecnologias

- JavaScript (ES6+)
- HTML5
- CSS3

## 📚 Conteúdo estudado

- Introdução ao JavaScript
- Variáveis e constantes
- Tipos de dados
- Operadores
- Estruturas condicionais
- Estruturas de repetição
- Funções
- Arrays
- Objetos
- Manipulação do DOM
- Eventos
- Métodos de Arrays
- Boas práticas de programação
- Organização de código

## 📂 Estrutura do repositório

> A organização das pastas acompanha a sequência dos módulos e exercícios desenvolvidos durante o curso.

## ⚙️ Como executar

A maioria dos projetos pode ser executada abrindo o arquivo `index.html` no navegador.

Caso utilize o VS Code, recomenda-se a extensão **Live Server** para facilitar o desenvolvimento.
